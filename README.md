# FRN-FM-Analyzer-v4.4
Назначение: Анализ и настройка ЧМ сигнала (девиация 3 кГц) в сети FRN (комната Test)
FRN FM Analyzer v4.4
Назначение: Анализ и настройка ЧМ сигнала (девиация 3 кГц) в сети FRN (комната Test).

Установка:
Установите Python 3.11 (обязательно с галочкой "Add to PATH")
Распакуйте архив FRN FM Analyzer v4.4.zip программы в папку,
запустите - installer.bat от имени администратора
(файлы installer.bat и fm_analyzer.py должны быть в одной папке)

Для старта программы запустите созданый после установки файл
start_frn_analyzer.bat

Основные функции
Калибровка:
Измеряет реальную девиацию радиостанции, рассчитывает корректирующий коэффициент
Генерация тестового сигнала:
Частоты 300, 1000, 3000 Гц, регулируемый уровень 0-100%
Сравнение сигналов:
Анализирует отправленный и принятый сигналы, вычисляет сходство в %
Спектральный анализатор:
В реальном времени показывает спектр входного сигнала (30-3000 Гц)
Автосохранение:
Логи и графики автоматически сохраняются в папку результатов

Калибровка (обязательна перед первым тестом)
Что нужно:
Радиостанция + компьютер
2 кабеля 3.5 mm jack + 2 конденсатора 0,47 мкФ
Громкость радиостанции и ПК: 50%
Линк переведён в комнату Test
Как работает:
Программа передаёт эталонный тон 1000 Гц
Измеряет реальную девиацию принятого сигнала
Рассчитывает коэффициент: К = 3000 / реальная_девиация
Коэффициент сохраняется и используется при тестах
Пример:
Реальная девиация 2400 Гц → коэффициент 1.25 (сигнал будет усилен на 25%)
Реальная девиация 3600 Гц → коэффициент 0.83 (сигнал будет ослаблен)

Подключение:
Компьютер (Line-Out) ──|| 0,47 мкФ ──→ Микрофон радиостанции
Компьютер (Line-In)  ←──|| 0,47 мкФ ─── Выход динамика радиостанции

Полученные результаты:
Временные метки, девиация, ошибка, сходство сигналов - log_дата.txt
Графики отправленного и принятого сигналов - comparison_дата.png

Быстрый старт:
1 - Подключите радиостанцию через конденсаторы 0,47 мкФ
2 - Установите громкость 50% на радиостанции и ПК
3 - Переведите линк в комнату Test
4 - Выберите звуковую карту (кнопка "Настройки звуковой карты")
5 - Нажмите "КАЛИБРОВКА" и удерживайте PTT
6 - Нажмите "НАЧАТЬ ТЕСТ" и удерживайте PTT
